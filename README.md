# Factorial Kata

[Factorial - Wikipedia]((http://en.wikipedia.org/wiki/Factorial)

From Wikipedia:

> In mathematics, the factorial of a non-negative integer n, denoted by n!, is
> the product of all positive integers less than or equal to n.


## Example

```
0! = 1

1! = 1

2! = 2 x 1 = 2

3! = 3 x 2 x 1 = 6
```
